O sistema utiliza:

- Árvore Binária de Busca (BST) para organização dos registros  
- Criptografia simétrica (Fernet) para proteger os dados salvos  
- Hash de senha mestre para autenticação  
- Proteção contra brute force com bloqueio progressivo  
- Persistência criptografada em arquivo  

Desenvolvido com objetivo educacional para praticar estruturas de dados, criptografia e arquitetura modular em Python, sinta-se livre para modificar.
